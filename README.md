# Overview

AQI Monitoring Platform is a modern environmental intelligence dashboard designed to monitor, visualize, and analyze air quality conditions through an interactive and responsive web interface.

The platform focuses on real-time AQI awareness, pollution analytics, environmental monitoring workflows, and scalable frontend visualization architecture.

This repository contains the frontend visualization layer of the AQI ecosystem.

---

# Core Features

- Real-time AQI visualization
- Pollution severity indicators
- Interactive environmental analytics dashboard
- Responsive UI optimized for desktop and mobile
- Modular frontend architecture
- Lightweight and fast deployment pipeline
- Dynamic chart rendering workflows
- Environmental condition monitoring system

# Development Workflow

Clone the repository:

```bash
git clone https://github.com/HariKumar-DS/aqi_monitoring_website_view.git
```

Move into the project directory:

```bash
cd aqi_monitoring_website_view
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

# Production Build

```bash
npm run build
```
